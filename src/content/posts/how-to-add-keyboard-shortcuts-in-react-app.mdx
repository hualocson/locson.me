## Basic Setup

You can use `useEffect` to listen for `keydown` events and handle shortcuts:

```tsx
useEffect(() => {
  const handleKeyPress = (event: KeyboardEvent) => {
    if (event.key === "a") {
      console.log("Pressed A");
    }
  };

  window.addEventListener("keydown", handleKeyPress);
  return () => {
    window.removeEventListener("keydown", handleKeyPress);
  };
}, []);
```

## Reusable Hook

To avoid repeating code, create a reusable useKeyPress hook:

```tsx
import { useEffect, useRef } from "react";

export function useKeyPress(
  keys: string[],
  callback: (event: KeyboardEvent) => void,
  node: HTMLElement | Document = document
) {
  const callbackRef = useRef(callback);

  useEffect(() => {
    callbackRef.current = callback;
  });

  useEffect(() => {
    const handleKeyPress = (event: KeyboardEvent) => {
      if (keys.includes(event.key)) {
        callbackRef.current(event);
      }
    };

    node.addEventListener("keydown", handleKeyPress);
    return () => {
      node.removeEventListener("keydown", handleKeyPress);
    };
  }, [keys, node]);
}
```

## Example Usage

```tsx
function App() {
  useKeyPress(["a"], () => {
    console.log("Shortcut A pressed");
  });

  useKeyPress(["s"], () => {
    console.log("Shortcut S pressed");
  });

  return <div>Try pressing A or S</div>;
}
```

## Summary

- Use `useEffect` for basic keydown listeners.

- Check modifier flags for combos like `Shift` + `A`.

- Extract logic into a <code className="text-red-400">{"useKeyPress"}</code> hook for reusability.

- Attach to document or specific elements for flexible control.
